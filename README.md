# Config
Contains my config for apps I use (Vim and Tmux) and some helpful scripts. 

# Install packages nvim, xclip, tmux.

```bash
sudo apt update
sudo apt install nvim, xclip, tmux
sudo apt install npm # Needed to install LSP servers
```

# Install user config

```bash
cp -f .tmux.conf ~/.
cp -rf nvim ~/.config/.
```
